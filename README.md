# Creative AI (CAI) - Offline AI assistant


CAI is a fast offline AI assistant with a modern GUI and memory system.

# Features:

- Fully offline
- Modern CustomTkinter interface
- Memory system
- File reading for coding files
- Fast responses
- Installer included
